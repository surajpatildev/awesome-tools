## 🤝 Contributing

Found an amazing tool that deserves to be here? I'd love to hear about it!

### How to Contribute

1. **Fork this repository**
2. **Add your tool** using this format:
   ```markdown
   - **[Tool Name](https://tool-url.com/)** - Brief description
     - *Use case: What problem it solves*
     - *Why it's great: What makes it special*
   ```
3. **Submit a pull request**

### Contribution Guidelines

- ✅ Tool must be actively maintained
- ✅ Should solve a specific problem well
- ✅ Include both use case and why it's great
- ✅ Test the link before submitting
- ✅ Place in the most appropriate category
- ❌ No affiliate links
- ❌ No paid promotions (genuine recommendations only)

## 📝 Template for New Entries

```markdown
- **[Tool Name](https://tool-url.com/)** - One-line description of what it does
  - *Use case: Specific problem it solves or workflow it improves*
  - *Why it's great: What makes it stand out from alternatives*
```
